# Proyecto final
## Aprendizaje de máquina
### Inteligencia Artificial — Facultad de Ciencias, UNAM

Desarrollar un proyecto que aplique aprendizaje de máquina para la resolución de un problema específico. Se seguirán los siguientes puntos:

1. **(1 pt)** Elegir un conjunto de datos adecuado que responda a un problema de clasificación, regresión o agrupamiento. Este conjunto de datos se puede tomar de páginas como Kaggle, HuggingFace o algún otro repositorio o paquetería.

2. **(1.5 pts)** Se debe describir de manera precisa el conjunto de datos: el número de datos totales, los datos usados para el entrenamiento y para la evaluación (si se hace validación también se debe señalar). Asimismo, se debe hacer una breve descripción estadística que muestre el rango de los valores, los valores medios, etc. Se puede usar la paquetería Pandas.

3. **(1.5 pts)** Se deberán elegir al menos dos modelos de aprendizaje de máquina orientado al problema (por ejemplo, para clasificación, se puede usar perceptrón y regresión logística).

4. **(3 pts)** Se deberán de describir detalladamente los modelos y los motivos de la selección, así como el régimen de entrenamiento: hiperparámetros, número de iteraciones, etc. El régimen de entrenamiento se debe llevar a cabo adecuadamente.

5. **(2 pts)** Se deberá reportar la evaluación con métricas precisas (precisión, exhaustividad, error cuadrático medio o el que sea adecuado para el caso).

6. **(1 pt)** Se debe incluir una sección de conclusiones donde se haga un análisis cualitativo de los resultados obtenidos.
